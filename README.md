# 🎮 Godot Practice — Shoot'em Up

A small practice project built in **Godot 4.6.1**, focused on developing a **basic shoot'em up**.

## 🧩 Implemented Features

### 🎮 Gameplay
- Player movement  
- Shooting mechanics  
- Enemy waves  
- Random generation of power-ups and asteroids  

### 🧱 Physics & Collisions
- Physics body and area handling  
- Group system usage  

### 👾 Enemies
- Automatic patrol and attack patterns  
- Enemy-to-enemy interactions  
- Player detection and chase behavior (if applicable)  

### 🖥️ UI & Scenes
- Health and score system  
- Options menu navigation  
- Scene switching  
- Node instancing and removal  
- `CanvasLayer` for proper UI layout  
- Responsive scaling for multiple resolutions  
- Custom UI themes  

### 🎨 Environment
- On-screen node activation/deactivation optimization  

### 💻 Code Architecture (GDScript)
- Clean, readable, and scalable code  
- Modular structure with reusable components  
- Clear separation of responsibilities  
- Global variable management  

+ Patterns:
  + Signal Hub  
  + Singleton (Autoloads)  

- Signals  
- Custom signals  
- Timers  

## 🛠️ Technologies
- Godot Engine 4.6.1  
- GDScript  
- Git (Version control)
